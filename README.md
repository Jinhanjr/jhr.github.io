## 🔮 About BaZi Creation

BaZi Creation is a BaZi (Four Pillars of Destiny) analysis app that provides deep insights into:

- Wealth & Financial Capacity
- Career & Professional Path
- Personality & Inner Nature
- Relationships & Synergy
- Life Stages & Fortune Cycles

It uses a combination of classical BaZi logic and AI to deliver personalised, consistent, and reliable reports.
